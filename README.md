# BaytAI

نسخة تشغيلية أولية لمنصة تحويل الشعر العربي إلى فيديو أو بودكاست.

## التشغيل محليًا

```bash
pip install -r requirements.txt
uvicorn app:app --reload
```

ثم افتح:

```text
http://127.0.0.1:8000
```

## ماذا تفعل هذه النسخة؟
- واجهة عربية كاملة
- إدخال القصيدة
- اختيار نوع المخرج والسياق والصوت
- توليد تحليل تلقائي ونموذج أولي للنتيجة

## كيف تصبح منتجة فعليًا؟
1. ربط OpenAI لتحليل القصيدة واستخراج السيناريو.
2. ربط ElevenLabs أو Azure TTS لإنتاج الصوت العربي.
3. ربط Runway أو Pika أو Stability لإنتاج الصور/الفيديو.
4. إضافة قاعدة بيانات Supabase لحفظ المشاريع والمستخدمين.
5. نشر التطبيق على Render أو Railway أو Vercel.

## ملاحظات
هذه الحزمة جاهزة كنقطة انطلاق حقيقية، لكنها لا تتضمن مفاتيح API ولا النشر الخارجي، لأن ذلك يحتاج حساباتك وبيانات الدفع الخاصة بك.
